## Intake rules

1. An **Area** is ongoing (no deadline). If it has a deadline, it's a Project (`01_Projects/`); if it's someone else's material, it's a Resource (`03_Resources/`).
2. New area = new folder + a row here + an area note using `System/templates/area-document-template.md`.
3. Apply Zettelkasten enrichment on filing: 3-5 `[[WikiLinks]]`, backlinks in connected notes.
4. Entity-like facts (`role`, `status`, `company`, `affiliation`) get bi-temporal `timeline` entries — see `.agent/steering/bi-temporal-tracking.md`.
