# Obsidian Flavored Markdown

Create and edit Obsidian-compatible Markdown files with all Obsidian-specific syntax extensions.

## When to Use
- Creating or editing `.md` files in the vault
- Working with wikilinks, callouts, embeds, or properties
- Formatting notes with Obsidian-specific syntax (highlights, comments, block IDs)
- Generating frontmatter/YAML properties for notes

## What It Covers
- Internal links (wikilinks): `[[Note Name]]`, `[[Note#Heading]]`, `[[Note#^block-id]]`
- Embeds: `![[Note]]`, `![[image.png|300]]`, `![[doc.pdf#page=3]]`
- Callouts: `> [!note]`, `> [!tip]- Foldable`, nested callouts
- Properties (YAML frontmatter): tags, aliases, cssclasses, custom fields
- Tags: `#tag`, `#nested/tag`
- Comments: `%%hidden%%`
- Math (LaTeX), Mermaid diagrams, footnotes
- Standard Markdown (CommonMark + GFM)
