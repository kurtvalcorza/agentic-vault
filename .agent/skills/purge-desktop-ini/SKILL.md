# Purge Desktop.ini Files

## Description
This skill executes a maintenance script to recursively find and delete `desktop.ini` files throughout the current workspace. These files are automatically generated by Windows and can clutter version control or file listings.

## Triggers
- "Purge desktop.ini"
- "Clean up desktop.ini files"
- "Remove windows artifacts"
- "Delete system config files"

## Actions

### 1. Execute Cleanup Script
Run the PowerShell script located in the agent scripts directory. This script handles the recursive deletion and error suppression.

```powershell
.agent/scripts/purge-desktop-ini.ps1
```

### 2. Report Results
- Confirm execution.
- detailed script output is available if needed, but a summary is sufficient (e.g., "Purged X files" or "No files found").
