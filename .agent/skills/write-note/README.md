# Write Note

Interactive atomic note creation through Socratic interrogation.

## When to Use
- Capturing a new concept or idea
- Deepening understanding of a topic
- Building knowledge graph connections
- Creating permanent notes from fleeting thoughts

## What It Does
1. Guides concept clarification through questions
2. Uses Socratic method to deepen understanding
3. Discovers connections to existing notes
4. Drafts atomic note with proper structure
5. Adds backlinks to related files

## Five-Phase Process

### 1. Concept Clarification
- What is the core idea?
- Why does it matter?
- What's the simplest explanation?

### 2. Socratic Interrogation
- Challenging assumptions
- Exploring edge cases
- Uncovering hidden implications
- Testing understanding

### 3. Connection Discovery
- Search existing notes for related concepts
- Identify potential links
- Map knowledge graph relationships

### 4. Note Draft
- One idea per note (atomic principle)
- Clear, concise writing
- Evidence and examples
- Proper metadata

### 5. Finalization & Linking
- Save to `Notes/[Note-Name].md`
- Add backlinks to related notes
- Update knowledge graph

## Atomic Note Principles
- **One Idea**: Single concept per note
- **Self-Contained**: Understandable without context
- **Heavily Linked**: Minimum 3-5 connections
- **Evergreen**: Timeless, not time-bound

## Quick Start
Just say: **"Let's write a note about [concept]"**

## Output Location
`Notes/[Note-Name].md` with automatic backlinks
