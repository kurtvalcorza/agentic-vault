# JSON Canvas

Create and edit JSON Canvas files (`.canvas`) with nodes, edges, groups, and connections.

## When to Use
- Creating or editing `.canvas` files in the vault
- Building visual canvases, mind maps, or flowcharts
- Connecting notes, files, and links on an infinite canvas
- Generating project boards or research canvases

## What It Covers
- JSON Canvas Spec 1.0 file structure (`nodes` + `edges`)
- Node types: `text` (Markdown), `file` (attachments), `link` (URLs), `group` (containers)
- Edge connections with sides, labels, and arrow endpoints
- Color presets (`"1"`–`"6"`) and hex colors
- Group backgrounds and styles (`cover`, `ratio`, `repeat`)
- Layout guidelines: positioning, sizing, spacing
- ID generation (16-char hex)
- Validation rules
