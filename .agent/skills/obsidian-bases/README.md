# Obsidian Bases

Create and edit Obsidian Bases (`.base` files) with views, filters, formulas, and summaries.

## When to Use
- Creating database-like views of vault notes
- Building `.base` files with table, cards, list, or map views
- Writing filter expressions, formulas, or custom summaries
- Configuring property display names and grouping

## What It Covers
- YAML-based `.base` file format and complete schema
- Filter syntax: `and`, `or`, `not`, nested conditions
- Formula expressions: arithmetic, conditionals, date math, string manipulation
- All built-in functions: date, string, number, list, file, link, object
- Duration type handling (date subtraction)
- View types: table, cards, list, map
- Summary formulas: Average, Sum, Min, Max, Median, etc.
- Embedding bases in Markdown: `![[MyBase.base]]`
