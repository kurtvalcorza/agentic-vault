# Optimize Workspace

Comprehensive workspace audit and optimization system for maintaining vault health and organizational clarity.

## What This Does

Performs a 4-phase analysis of your entire vault:

1. **Discovery & Mapping**: Catalogs all skills, agents, outputs, and dependencies
2. **Quality Assessment**: Checks documentation completeness, naming conventions, broken links
3. **Redundancy & Gap Analysis**: Identifies duplicate capabilities and missing functionality
4. **Strategic Recommendations**: Provides actionable optimization plan with prioritized tasks

## When to Use

- **Quarterly maintenance**: Keep vault organized and efficient
- **After major projects**: Clean up accumulated outputs and temporary files
- **Skill proliferation**: When you have many skills and need clarity on what exists
- **Onboarding**: Get comprehensive overview of workspace capabilities
- **Performance issues**: Identify bloat, orphaned files, or organizational debt

## What You Get

Four reports saved to `.agent/outputs/`:

- **workspace-map.md**: Complete inventory of skills, agents, outputs, dependencies
- **quality-report.md**: Documentation gaps, naming violations, broken links
- **dependency-analysis.md**: Skill relationships, redundancy matrix, orphaned files
- **optimization-plan.md**: Prioritized recommendations (critical/high/medium/low)

## How to Trigger

Simply say: **"Optimize my workspace"** or **"Run workspace optimization"**

The agent will execute all four phases automatically and provide a summary with links to detailed reports.

## Best Practices

- Run quarterly to maintain workspace health
- Review optimization plan before implementing recommendations
- Archive obsolete skills rather than deleting (preserve institutional knowledge)
- Use findings to inform skill consolidation and refactoring decisions
