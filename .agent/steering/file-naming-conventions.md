## File Naming Conventions

### Standard Formats
*   **Daily notes**: `YYYY-MM-DD.md` (e.g., `2026-02-02.md`)
*   **Skills**: `skill-name/SKILL.md` and `skill-name/README.md` (stored in `.agent/skills/`)
*   **Skill Registries**: `SKILLS-REGISTRY.md` (standardized across all agent workspaces)
*   **Prompts**: `category_prompt-purpose.md`
*   **Outputs**: `YYYY-MM-DD_Output-Description.md`
*   **Projects**: `PascalCase` or `kebab-case` (e.g., `Project Beacon`, `atlas-dashboard`)

### Linking Standards
*   **WikiLinks**: Use `[[Note Name]]` for internal connections
*   **Aliases**: `[[Actual Name|Display Name]]` for readability
*   **Minimum Links**: Aim for 3-5 links per note to build knowledge graph
