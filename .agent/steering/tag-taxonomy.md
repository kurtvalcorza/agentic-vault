## Tag Taxonomy

### Standard Tags
*   **Agents**: `#agent`, `#agent/utility`, `#agent/validator`
*   **Skills**: `#skill`, `#skill/[domain]`, `#skill/[specialization]`
*   **Projects**: `#project/[name]` (e.g., `#project/beacon`, `#project/atlas`)
*   **Status**: `#status/active`, `#status/completed`, `#status/archived`, `#status/experimental`
*   **Domains**: `#research`, `#presentation`, `#reporting`, `#synthesis`

### Usage Guidelines
*   Use lowercase for consistency
*   Prefer hierarchical tags (`#skill/research` over `#research-skill`)
*   Tag skills and agents for discoverability
*   Mark experimental features explicitly
